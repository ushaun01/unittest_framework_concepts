🧪 Python Unittest – Concept Learning Repository

📌 Overview

This repository contains my hands-on practice while learning the Python unittest framework.
It focuses on understanding core testing concepts through simple and clear examples.

This is a learning-focused repository, not a real-world project.


🎯 Purpose

* To understand how the unittest framework works
* To practice writing and executing test cases
* To explore different types of assertions and fixtures



🧠 Concepts Practiced

* Creating test cases using `unittest.TestCase`
* Running tests using command line
* Assertions:

  * assertEqual
  * assertNotEqual
  * assertTrue / assertFalse
* Setup & Teardown:

  * `setUp()` / `tearDown()`
  * `setUpClass()` / `tearDownClass()`
  * `setUpModule()` / `tearDownModule()`
* Skipping tests
* Basic test suite execution
* Organizing test files using packages

 How to Run

Run all tests:
python -m unittest discover

Run a single file:
python filename.py
